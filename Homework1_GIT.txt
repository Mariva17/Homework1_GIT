1. 
а). Проверить наличие прикрепленного SSH-ключа в GITHub. Еще можно проверить наличие user name и email в терминале( GITBush) git config --global -l,
 тк должны быть зарегистрированы.
б). В репозитории заходим на вкладку CODE, выбираем SSH-протокол, копируем.
в). В GITBush перейти в ту директорию, куда будем копировать репозиторий и вставить ссылку. Создастся директория с названием репозитория.

2.
а).В GITBush перейти в ту директорию, которую будем отправлять на GitHub.
б). git init
в). git add .(если несколько файлов в папке или название файла)
г). git commit -m "комментарий"
д). git remote add origin...
е). git branch -M main
ж). git push -u origin main

3.
git pull

4.
git status - показывает состояние репозитория (есть ли файлы для добавления в репозиторий и пр.)
git log - показывает историю действий с репозиторием
git remote -v - показывает связку с удаленным репозиторием
